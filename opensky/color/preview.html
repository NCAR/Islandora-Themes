Atmos Theme Color Selector

The color scheme for the Atmos theme can be set here. Remember that blocks only appear in pages that you set them to appear in. It is unlikely that you will have a page displaying all of the blocks shown on this page at the same time.

Once you modify and save your settings, most changes will have an immediate effect. Some color changes to blocks will require that you set a CSS class name when editing the block. Available block classes are:

  • lightblock
  • mediumblock
  • darkblock

More information is available in the Atmos Theme README.txt file. The README file can be found in the Atmos theme folder.

More information regarding customizing the logo and site slogan can be found in the README-logo.txt & README-slogan.txt files. These file can be found in the psd folder located inside of the main Atmos theme folder.
